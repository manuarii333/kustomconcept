⚙️ Guide de démarrage — Agent Vocal
1
Installer les dépendances Python
Depuis le dossier agents/, installer les packages requis.
pip install flask anthropic openai telnyx requests python-dotenv
2
Configurer le fichier .env
Créer agents/.env avec les clés API Telnyx, OpenAI, Anthropic, Airtable et SMTP.
ANTHROPIC_API_KEY=sk-ant-...  |  OPENAI_API_KEY=sk-...  |  TELNYX_API_KEY=KEY...
3
Démarrer le serveur Flask (port 5002)
python agents/agent_vocal_client.py
4
Exposer avec ngrok (pour Telnyx webhook)
Telnyx a besoin d'une URL publique pour envoyer les événements d'appel.
ngrok http 5002
→ Copier l'URL https://xxxx.ngrok.io dans la console Telnyx > Call Control > Webhook URL
→ Ajouter /telnyx/appel-entrant à la fin.
5
Configurer Telnyx
Dans la console Telnyx :
1. Créer une Call Control Application
2. Webhook URL → https://xxxx.ngrok.io/telnyx/appel-entrant
3. Affecter le numéro HCS à cette application
4. Copier le Connection ID dans le .env (TELNYX_CONNECTION_ID)
✓
Tester l'agent sans appel
Utiliser le panneau "Tester l'agent" ci-dessus, ou directement dans le navigateur :
http://localhost:5002/test-voix?msg=Bonjour
