🚀 Démarrage rapide
Première découpe en 5 étapes — de l'ouverture du logiciel à la pièce découpée
1
Lancer SignMaster PRO
Double-cliquer sur l'icône bureau ou Démarrer → SignMaster PRO. Attendre le chargement complet de l'interface (barre d'outils + zone de travail visibles).
💡 Si le logiciel demande une clé de licence au démarrage, elle est sur l'autocollant du CD original ou dans le dossier C:\HCS\Licences\
2
Créer ou importer un document
Nouveau : Fichier → Nouveau — définir largeur = largeur de votre rouleau (ex. 500 mm) et hauteur libre.
Importer : Fichier → Importer ou glisser-déposer un fichier SVG/AI/EPS/PDF directement dans la zone de travail.
💡 Toujours vérifier l'unité (mm) dans Options → Paramètres document avant de commencer.
3
Préparer les tracés de découpe
Sélectionner tous les objets à découper. Vérifier que les chemins sont des contours fermés (pas de remplissage uniquement). Utiliser Objet → Vectoriser si besoin pour convertir un bitmap.
Affecter la couleur Registration (noir pur) aux lignes de coupe.
💡 Pour le vinyle flocage : prévoir au minimum 5 mm de marge entre les lettres pour faciliter le désherbage.
4
Configurer les paramètres de coupe
Cliquer sur l'icône ✂️ Envoyer au traceur dans la barre d'outils (ou Fichier → Découper/Tracer).
Dans la fenêtre qui s'ouvre : sélectionner votre modèle de plotter, régler Vitesse et Pression selon le matériau (voir section Matériaux).
💡 Toujours faire un test de coupe sur un coin du vinyle avant de lancer la découpe complète.
5
Lancer la découpe & désherber
Cliquer sur Envoyer. Le plotter commence à découper. Une fois terminé :
1. Retirer le rouleau du plotter
2. Couper la pièce avec le couteau rotatif
3. Désherbager (retirer le vinyle excédentaire) avec le crochet de désherbage
4. Appliquer le masque de transfert si nécessaire
💡 Si les coins sont arrondis ou si la lame saute, consulter la section Dépannage.

✅ Bonnes pratiques design

• Travailler en mm dès Illustrator
• Vectoriser tout texte (Texte → Vectoriser)
• Supprimer les contours doubles avant export
• Nommer les calques : COUPE / IMPRESSION
• Garder une copie source en .AI avant export SVG

⚠️ Points d'attention HCS

• Toujours vérifier le sens d'enroulement du vinyle (lisse vers le bas sur le plotter)
• Pour le flocage : miroir horizontal obligatoire (Objet → Retourner H)
• Taille minimale texte lisible : 8 mm de hauteur en vinyle, 12 mm en flocage
• Archiver le fichier SignMaster dans Dropbox /HCS/2026/Production/

🎯 Optimisation mise en page

• Regrouper les pièces similaires sur la même bande (Édition → Imbriquer)
• Laisser 10 mm de marge en haut du vinyle (bord de saisie plotter)
• Trier par couleur pour minimiser les changements de vinyle
• Utiliser Répéter (Ctrl+D) pour les séries

📦 Après découpe

• Étiqueter chaque pièce avec nom client + N° commande Odoo
• Stocker les restes de vinyle roulés (pas aplatis)
• Logger la découpe dans Agent 4 Stock (consommation vinyle)
• Scanner les pièces terminées avec Agent 9 OCR si facture fournisseur

✂️ Contour Cut (Print & Cut)
Découpe après impression — pour stickers, étiquettes, badges personnalisés
01
Préparer le fichier
Créer deux calques : IMPRESSION (couleurs) et COUPE (contour noir). Le calque COUPE définit le tracé de découpe autour du motif imprimé.
02
Ajouter les marques de repérage
Dans SignMaster : Découpe → Ajouter marques ARMS. Minimum 3 marques, une en haut-gauche, une en haut-droite, une en bas-gauche.
03
Imprimer avec les marques
Imprimer le calque IMPRESSION + les marques via votre imprimante. Les marques doivent être visibles et non coupées sur la feuille imprimée.
04
Charger dans le plotter
Placer la feuille imprimée dans le plotter. S'assurer qu'elle est bien droite et maintenue par les rouleaux de pression.
05
Calibration ARMS / Laser
Dans SignMaster : Découpe → Calibrer marques. Le plotter scanne les marques automatiquement. Si laser pointer : ajuster le décalage X/Y dans les paramètres plotter.
06
Lancer la découpe de contour
Cliquer Envoyer au traceur en sélectionnant uniquement le calque COUPE. Vérifier que le mode Contour Cut est activé dans les options.
🔧 Réglage offset lame pour Contour Cut

L'offset lame (blade offset) compense la distance entre la pointe de la lame et son axe de rotation.
Valeur standard : 0.25 mm pour lame 45° · 0.5 mm pour lame 60°
Pour calibrer : découper un carré de 20×20 mm et mesurer. Si les coins sont arrondis, augmenter l'offset de 0.05 mm.

⚙️ Configuration du plotter
Ajouter et configurer votre traceur dans SignMaster PRO
1
Brancher le plotter
Connecter en USB ou en série (RS-232). Allumer le plotter AVANT de lancer SignMaster. Sur Windows : vérifier que le port apparaît dans Gestionnaire de périphériques → Ports (COM et LPT). Noter le numéro de port COM (ex: COM3).
2
Ajouter le plotter dans SignMaster
Aller dans Configuration → Paramètres plotter → Ajouter. Dans la liste des modèles, chercher votre marque. Si votre modèle exact n'est pas listé, choisir le modèle le plus proche de même marque, ou sélectionner Generic HPGL/2 comme base.
3
Configurer le port de communication
Sélectionner le port : USB (recommandé) ou COM3 (série). Pour le port série : vitesse 9600 ou 38400 bauds selon votre plotter. Bits de données : 8 · Parité : Aucune · Bits d'arrêt : 1.
4
Définir la largeur de découpe
Entrer la largeur utile de votre plotter (entre les roulettes de pression). Exemple : pour un plotter 630 mm, entrer 580 mm (laisser 25 mm de chaque côté). Cette valeur définit la zone de travail dans SignMaster.
5
Test de connexion
Cliquer Tester la connexion. Le plotter doit émettre un bip ou bouger légèrement la tête. Si pas de réaction : vérifier le câble, relancer SignMaster, ou réinstaller le driver USB depuis le CD du plotter.
6
Test de coupe
Charger un bout de vinyle. Dans la fenêtre découpe, cliquer Test de coupe. SignMaster trace un petit triangle dans un cercle. Si la découpe est nette, la configuration est correcte. Ajuster l'offset lame si les coins sont arrondis.
